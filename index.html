IPv4 Address Information

  • Class A: 1.0.0.0 to 126.255.255.255 (Default Mask: /8) - Used for very large networks.
  • Class B: 128.0.0.0 to 191.255.255.255 (Default Mask: /16) - Used for medium to large networks.
  • Class C: 192.0.0.0 to 223.255.255.255 (Default Mask: /24) - Used for small networks.
  • Class D: 224.0.0.0 to 239.255.255.255 - Reserved for Multicast traffic (one-to-many communication).
  • Class E: 240.0.0.0 to 255.255.255.255 - Reserved for experimental or future use; should not be used on the public internet.
  • Loopback: 127.0.0.0 to 127.255.255.255 (commonly 127.0.0.1, known as `localhost`) - Reserved range used by a host computer to send network traffic to itself. Essential for testing network software and services without sending data over a physical network interface.

Note: Classful addressing is largely historical. Classless Inter-Domain Routing (CIDR) is the standard used today for more flexible IP address allocation.
